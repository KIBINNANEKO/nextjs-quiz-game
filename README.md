# Valorant Characters Documentation

## Technology stack:

- HTML, SCSS-modules;
- JavaScript;
- React;
- Next.js;
- MobX (state manager);
- Figma (design).

## How to start?

1. Type command **yarn install** in a console;
2. Type command **yarn dev/build** in a console;
3. Open [localserver](http://localhost:3000) with your browser;
